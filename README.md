# PilaEngine

Это мой собственный 2D-игровой движок, который я пишу на C++ с использованием графики OpenGL. Главная идея проекта — разобраться, как устроена графика на низком уровне, и собрать удобный каркас для создания двухмерных игр.

## 🎯 Цели проекта
* **Удобная работа с графикой:** Сделать так, чтобы не нужно было писать кучу сырого кода OpenGL для каждого объекта. Всё упаковывается в удобные C++ классы.
* **Быстрый 2D-рендеринг:** Сделать движок оптимизированным для плавной отрисовки картинок, спрайтов и фигур.
* **Скрипты на Lua:** Встроить поддержку языка Lua, чтобы можно было писать логику игры и настраивать уровни без постоянной пересборки самого движка.
* **Только 2D:** Проект полностью сосредоточен на двухмерной графике. Переход в 3D если и случится, то только в очень далеком будущем.

## 🚀 Что уже работает
* **Управление буферами:** Написаны C++ обертки для управления памятью видеокарты (классы VAO, VBO, EBO). Они сами следят за выделением и очисткой памяти.
* **Загрузка шейдеров:** Написан класс `Shader`, который сам читает файлы кода для видеокарты (`.glsl`), компилирует их и выводит ошибки в консоль, если в коде шейдера что-то не так.
* **Класс отрисовщика:** Создан отдельный класс `renderer`, который отвечает за вывод объектов на экран и очистку кадра.
* **Стабильный игровой цикл:** Движок запускает окно, следит за событиями и строго держит 60 кадров в секунду

## 🛠 Инструменты
* **Язык:** C++17
* **Графика:** OpenGL 3.3
* **Библиотеки:** GLFW3 (окно и ввод), GLAD (функции OpenGL), GLM (математика для графики)
* **Сборка:** CMake
